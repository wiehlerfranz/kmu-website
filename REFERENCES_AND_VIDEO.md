# Referenzen & Video – Anleitung

## 🎬 Videoskript (60–90 Sekunden)

*[Ruhig in Kamera blicken]*

„Ich bin Franz Wiehler, Konfliktnavigator und ehemaliger CFO von Siemens Sub-Sahara Afrika.

In meinen 30 Jahren Führungserfahrung habe ich gelernt: Konflikte entstehen, wenn unterschiedliche Interessen aufeinanderprallen. Das ist normal. Das Problem ist nicht der Konflikt – das Problem ist, wenn niemand ihn löst.

Genau da helfe ich. Ich höre allen Seiten zu. Nicht um recht zu haben, sondern um zu verstehen. Dann bringe ich Klarheit in die Sache und Struktur in den Prozess. So entstehen Lösungen, mit denen alle leben können.

Ob Nachfolge, Krise, Verhandlung oder Teamkonflikt – mein Ansatz ist immer der gleiche: Vertraulich, direkt, lösungsorientiert.

Wenn du merkst, dass eine Situation festgefahren ist: Lass uns sprechen. Ein erstes Telefonat kostet dich nichts außer 20 Minuten Zeit – und könnte alles verändern.

Deine Kontaktdaten findest du unten."

*[Ruhiger Ende-Blick]*

---

### Produktionstipps:
- **Länge:** ~1:00–1:30 (sprechen Sie langsam und deutlich)
- **Hintergrund:** Neutral, hell, ohne Ablenkung
- **Licht:** Tageslicht frontal oder indirekt (keine Gegenlicht-Silhouette)
- **Kamera:** Auf Augenhöhe, stabil (Stativ/Beanie Bag)
- **Audio:** Externes Mikrofon, ruhige Umgebung
- **Format:** Quer aufnehmen (Landscape, 16:9)
- **Kompression:** MP4, H.264, ≤100 MB (Zieldatei)
- **Optional:** Untertitel hinzufügen (ideal für LinkedIn & Mobilgeräte)

## 📝 Referenzen – Template

Sammel **2–4 verbindliche Aussagen** mit Namen und Kontext. Pro Referenz:

### Referenz-Template

```
Zitat: „[1–2 Sätze, prägnante vollständige Aussage – kein Fragment]"
Name: [Vollname oder „Geschäftsführer"]
Rolle: [z. B. „Geschäftsführer", „Gründer & CEO"]
Kontext: [z. B. „Familienunternehmen, Maschinenindustrie, 100 Mitarbeitende"]
```

### Beispiele (zum Anpassen)

**Referenz 1: Nachfolge-Erfolg**
```
Zitat: „Franz führte eine festgefahrene Nachfolge in sechs Wochen zu einem klaren gemeinsamen Plan. Sein Schutzraum tat der ganzen Familie gut."
Name: [Dein Kontakt]
Rolle: Gesellschafter / Inhaber
Kontext: Familienunternehmen, Maschinenbau
```

**Referenz 2: Verhandlung / Investment**
```
Zitat: „Er bereitete unsere Investorenrunde so vor, dass wir ruhig verhandeln konnten – am Ende standen 15% bessere Konditionen auf dem Tisch."
Name: [Dein Kontakt]
Rolle: Gründer & CEO
Kontext: Tech-Startup, Serie B
```

**Referenz 3: Interkulturell**
```
Zitat: „Der Blick aus Afrika, Asien und Deutschland zugleich half uns, kulturelle Stolperfallen rechtzeitig zu erkennen."
Name: [Dein Kontakt]
Rolle: Geschäftsführer
Kontext: Sozialer Träger, 100 Mitarbeitende
```

**Referenz 4: Team / Krise**
```
Zitat: „[Deine Aussage hier]"
Name: [Dein Kontakt]
Rolle: [z. B. Teamleiter, Betriebsrat]
Kontext: [Kontext]
```

## 🎥 Video hochladen

1. **Datei speichern als:** `self-portrait.mp4` im Ordner `assets/Videos/`
   (Falls der Ordner nicht existiert, erstelle ihn: `mkdir assets/Videos`)

2. **Im Code ist bereits vorbereitet:** `<video>` mit src auf `assets/Videos/self-portrait.mp4`

3. **Nach dem Hochladen:**
   ```bash
   git add assets/Videos/self-portrait.mp4
   git commit -m "Add self-portrait video"
   git push origin main
   ```
   Netlify deployt die Video-Datei dann automatisch.

4. **Live testen:** https://wiehler-kmu.de#video
